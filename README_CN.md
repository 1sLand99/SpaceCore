
# SpaceCore虚拟引擎内核  
  
SpaceCore是一个虚拟引擎内核，提供在Android上克隆和运行虚拟应用程序，并且已支持设备模拟、模拟定位以及其他更多的功能。本内核重新实现了现有框架的痛点，更贴合系统真实运行环境。

## 官网
[https://spacecore.dev/](https://spacecore.dev/)

## 兼容
Android 6.0 ～ 13.0 及 鸿蒙
